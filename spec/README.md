# Cursor Mirror 開発者仕様

この文書群は Cursor Mirror の規範仕様を定義する。対象には、オーバーレイの振る舞い、Windows 統合、配布条件、適合要件、テスト契約を含む。この文書はユーザーガイドではない。

この文書における `MUST`、`MUST NOT`、`SHOULD`、`MAY` は、すべて大文字で書かれている場合に限り、BCP 14 (RFC 2119 および RFC 8174) に従って解釈する。
