# Step 00 - Calibrator Verification Lab Data

## Purpose

Create a new MotionLab package specifically for Calibrator closed-loop verification. Existing trace, motion-recording, and calibration capture packages are not inputs to this POC because they may be deleted and because this experiment needs controlled scenarios.

## Generated Package

- Script: `scripts/new-calibrator-verification-lab-data.ps1`
- Package: `lab-data/calibrator-verification-v22.zip`
- Manifest: `lab-data/calibrator-verification-v22.manifest.json`
- Summary: `lab-data/calibrator-verification-v22.summary.md`

The package is ignored by git as generated data, but the generator and manifest/summary document how it was produced.

## Scenario Coverage

The scenario set is 50 seconds long at 240 Hz and includes:

- stationary/hold floor checks;
- constant medium horizontal motion;
- fast horizontal motion with late stop;
- reverse S-curve motion;
- fast diagonal motion;
- low-speed curved motion;
- repeated stop/resume motion;
- abrupt stop near the endpoint;
- micro-adjustments;
- mixed-speed wide sweep.

## Rule

All v22 Calibrator measurements must use this generated package or a later package generated by the same POC workflow. Repository-root `cursor-mirror-calibration-*.zip`, `cursor-mirror-motion-recording-*.zip`, and `cursor-mirror-trace-*.zip` files are excluded from scoring inputs.
